# Ethan Mathew Portfolio Site

Static portfolio website for Ethan Mathew.

## Local preview

Run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

This repository is configured for GitHub Pages deployment through GitHub Actions.

For the cleanest free URL, create a GitHub repository named:

`<your-github-username>.github.io`

Then push this project to that repository and enable GitHub Pages with the
source set to `GitHub Actions`.

Free site URL:

`https://<your-github-username>.github.io`

If you use a different repository name, the site URL will be:

`https://<your-github-username>.github.io/<repo-name>/`
